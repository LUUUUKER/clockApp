## 安装指南

1. 克隆仓库

    ```bash
    git clone https://github.com/你的用户名/项目名.git
    ```

2. 进入项目目录

    ```bash
    cd 项目名
    ```

3. 创建虚拟环境

    ```bash
    python -m venv venv
    ```

4. 启动虚拟环境

    - **Windows**:
        ```bash
        .\venv\Scripts\activate
        ```

    - **macOS/Linux**:
        ```bash
        source venv/bin/activate
        ```

5. 安装依赖

    ```bash
    pip install -r requirements.txt
    ```
